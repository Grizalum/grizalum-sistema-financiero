## ⚡ Instalación Rápida

### **Opción 1: Usar GitHub Pages (Recomendado)**

1. **Acceder directamente** al sistema en vivo:
   ```
   https://grizalum.github.io/grizalum-sistema-financiero/
   ```

2. **Instalar como PWA** (Opcional):
   - En Chrome/Edge: Clic en ⚙️ → "Instalar GRIZALUM"
   - En móvil: "Agregar a pantalla de inicio"

### **Opción 2: Clonar y Ejecutar Localmente**

```bash
# 1. Clonar el repositorio
git clone https://github.com/Grizalum/grizalum-sistema-financiero.git

# 2. Navegar al directorio
cd grizalum-sistema-financiero

# 3. Abrir con servidor local (opcional)
# Con Python 3:
python -m http.server 8000

# Con Node.js:
npx serve .

# 4. Abrir en navegador
# http://localhost:8000
```

### **Opción 3: Descargar ZIP**

1. **Descargar** desde GitHub: `Code` → `Download ZIP`
2. **Extraer** y abrir `index.html` en navegador
3. **¡Listo!** Sistema funcionando al 100%

## 💻 Uso del Sistema

### **🏠 Dashboard Principal**

Al ingresar al sistema verás:

- **KPI Cards Animadas** - Métricas financieras principales
- **Gráficos Interactivos** - Tendencias y comparativas
- **Sidebar de Navegación** - Acceso a todos los módulos
- **Indicadores en Tiempo Real** - Datos actualizados automáticamente

### **🧭 Navegación**

#### **Teclado (Atajos):**
- `Ctrl + 1` - Dashboard principal
- `Ctrl + 2` - Flujo de caja
- `Ctrl + N` - Nueva transacción
- `Ctrl + E` - Exportar datos
- `Esc` - Cerrar formularios

#### **Interfaz:**
- **Sidebar** - Menú principal organizado por categorías
- **Header** - Controles de período y notificaciones
- **Mobile** - Menú hamburguesa responsive

### **📊 Flujo de Caja**

#### **Agregar Transacción:**
1. Clic en **"Nueva Transacción"**
2. Completar formulario:
   - Fecha, tipo, categoría
   - Descripción y monto
   - Entrada o salida
3. **Guardar** - Se actualiza automáticamente

#### **Exportar Reportes:**
- **PDF** - Reporte profesional listo para imprimir
- **Excel** - Datos para análisis adicional
- **Gráficos** - Imágenes de alta resolución

## 🎨 Personalización

### **🎨 Colores y Temas**

El sistema utiliza variables CSS para fácil personalización:

```css
:root {
  /* Colores principales */
  --peru-gold: #d4af37;      /* Oro peruano */
  --primary-dark: #0f172a;   /* Azul corporativo */
  --profit: #059669;         /* Verde ganancias */
  --loss: #dc2626;           /* Rojo pérdidas */
}
```

### **🏢 Personalización Empresarial**

```javascript
// config/app-config.js
const appConfig = {
  company: {
    name: "Tu Empresa SAC",
    logo: "assets/images/tu-logo.png",
    currency: "S/.",
    locale: "es-PE"
  },
  theme: {
    primaryColor: "#d4af37",
    accentColor: "#0f172a"
  }
};
```

### **📊 Datos Iniciales**

Edita `src/data/sample-data.json` para cargar tus datos:

```json
{
  "company": "Tu Empresa",
  "initialCashFlow": 50000,
  "targetRevenue": 100000,
  "fiscalYear": 2025
}
```
